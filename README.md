# VAST-Challenge

Построение круговой диаграммы для оценки интенсивности обмена сообщениями в парке развлечений в промежутках дня длиной по 15 минут.
csv данные подгружаются в коллаб из этого репозитория
Для построения диаграммы Джона Сноу требуются координаты каждой из локаций парка развлечений.
Это проблема, потому что их много, а распознование с помощью tesseract ocr получилось говно.
Мб координаты есть где-то в файлах соревнования, но я прошляпила и не увидела, пока так.
